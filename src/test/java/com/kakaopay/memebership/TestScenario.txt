1. 회원 가입
- 테스트에 필요한 고객 정보를 생성한다.
  ㄴ 회원, 가족관계, 친구관계

- 테스트에 필요한 가맹점 정보를 생성한다.

#################################################################################
2. 바코드 발급
- 회원 ID 를 이용하여 유니크한 바코드를 발급 받는다.


#################################################################################
3. 발급된 바코드를 통한 포인트 적립
- 바코드 번호를 통해 등록되는 포인트 정보를 누적 시킨다. (DB)

#################################################################################
4. 바코드 사용
- 바코드 번호를 통해 지불 요청이 온 경우 이를 수행 한다.
  ㄴ 단 복수명의 사용자가 동시에 사용 할 수도 있으므로, 병행성을 보장해야 한다.
